Documentation

Delete a hash table.

The hash table has to be empty!

Parameters:
h - Hash table to delete
Returns:
int ret == 0: Ok.
int ret == -1: An error, see pbl_errno:
PBL_ERROR_EXISTS - The hash table is not empty.
