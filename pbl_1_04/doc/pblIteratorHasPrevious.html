int pblIteratorHasPrevious

( PblIterator* iterator )

Returns true if this iterator has more elements when traversing the collection in the reverse direction.

Documentation

Returns true if this iterator has more elements when traversing the collection in the reverse direction.

In other words, returns a value > 0 if pblIteratorPrevious would return an element rather than returning (void*)-1.

This method has a time complexity of O(1).

Parameters:
iterator - The iterator to check the previous element for
Returns:
int rc > 0: The iterator has more elements.
int rc == 0: The iteration has no more elements.
int rc < 0: An error, see pbl_errno:
PBL_ERROR_CONCURRENT_MODIFICATION - The underlying collection was modified concurrently.
